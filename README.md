# 🎬 Movie Recommender System

A simple movie recommendation web app built using Python and Streamlit.  
It suggests movies based on user input.

---

## 🚀 Features

- Search for a movie
- Get top recommended movies
- Simple and clean UI
- Fast recommendations

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn

---

## 📂 Project Structure

movie-recommender/
│── app.py
│── requirements.txt
│── movies.pkl
│── similarity.pkl
│── README.md

---

## ▶️ How to Run Locally

1. Clone the repository
   git clone https://github.com/hariganesh-git/movie-recommender.git

2. Go to folder
   cd movie-recommender

3. To get .pkl files
   run model.py

5. Install dependencies
   pip install -r requirements.txt

6. Run the app
   streamlit run app.py
