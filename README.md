# 🎬 Movie Recommender System

A simple movie recommendation web app built using Python and Streamlit.  
It suggests movies based on user input.

---

## 🚀 Features

- Search for a movie
- Get top recommended movies
- Simple and clean UI
- Fast recommendations

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn

---

## ▶️ How to Run Locally

1. Clone the repository
   git clone https://github.com/hariganesh-git/movie-recommender.git

2. Go to folder
   cd movie-recommender

3. For Cosine Similarity Pickle File 
   Due to GitHub size limits, download the similarity matrix from Google Drive:
   https://drive.google.com/uc?id=1iAMTPxBZ9aQuzXseuD0k2P4ymiq8V-LM

4. Install dependencies
   pip install -r requirements.txt

5. For Movie Poster
   create account in tmdb > generate API for personal use > use the api key terminal using set TMDB_API_KEY=your_api_key_here

5. Run the app
   streamlit run app.py
